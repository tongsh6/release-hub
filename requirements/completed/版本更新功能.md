# 需求：版本更新功能

> 状态：已完成
> 归档变更：`openspec/changes/archive/2026-02-04-add-version-updater/`

## 背景

ReleaseHub 需要自动化版本更新功能，支持在发布窗口执行时自动更新代码仓库中的版本号。

## 目标

- 支持 Maven 和 Gradle 项目的版本自动更新
- 提供版本校验和推导功能
- 集成到 Release Window 执行流程

## 已完成功能

1. ✅ **版本更新器**：Maven 和 Gradle 版本更新器实现
2. ✅ **版本更新 API**：单个和批量版本更新 API
3. ✅ **版本校验 API**：版本推导和格式校验
4. ✅ **前端集成**：版本更新对话框、Diff 查看、运行记录展示
5. ✅ **测试覆盖**：单元测试和集成测试（13 个测试用例）
