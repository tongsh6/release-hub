# 需求：添加版本号自动更新功能

## 背景
版本号自动管理是 ReleaseHub 的核心能力之一。为了减少人工修改版本号的错误与发布成本，需要支持 Maven/Gradle 项目版本号的自动更新，并将执行结果沉淀为可追踪的运行记录。

## 目标
- 支持 Maven/Gradle 版本号自动更新
- 支持单仓库与批量版本更新
- 支持执行结果与 diff 展示

## 验收标准
- [x] 支持 Maven pom.xml 版本更新
- [x] 支持 Gradle gradle.properties 版本更新
- [x] 支持版本更新 API 与运行记录落库
- [x] 支持前端触发与结果展示
