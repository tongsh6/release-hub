## 执行步骤

### Step 1: 安装依赖
```bash
pnpm add -D @pact-foundation/pact
# backend pom.xml + pact-jvm-provider-spring
```

### Step 2: Consumer（前端 → 生成 pact file）
```typescript
// src/api/__tests__/pact/auth.pact.spec.ts
const provider = new PactV3({
  consumer: 'releasehub-frontend',
  provider: 'releasehub-backend',
  dir: './pacts',
});

provider
  .given('admin user exists')
  .uponReceiving('a login request')
  .withRequest({ method: 'POST', path: '/api/v1/auth/login', ... })
  .willRespondWith({ status: 200, body: { token: '...' } });
```

### Step 3: Provider（后端 → 验证 pact file）
```java
@Provider("releasehub-backend")
@SpringBootTest(webEnvironment = RANDOM_PORT)
class AuthPactIT {
    @TestTemplate
    @ExtendWith(PactVerificationInvocationContextProvider.class)
    void verify(PactVerificationContext ctx) { ctx.verifyInteraction(); }
}
```

### Step 4: VERIFY
- `pnpm test:pact` consumer 合约生成
- `mvn test` provider 验证通过
